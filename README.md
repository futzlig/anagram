# Anagram
A small game to check if one word is an [anagram](https://en.wikipedia.org/wiki/Anagram) of another.

Here's a small list of anagrams:

* silent, listen
* a gentleman, elegant man
* eleven plus two, twelve plus one
* post, pots, spot, stop
* bier, brei, brie, rieb (german words)

## Compile and Run the Programm
Your need Maven and Java JDK 21 or higher. To compile and run the program enter the following
command:

    mvn compile exec:java

Have fun.
